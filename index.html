HEXO+GITHUB做个人博客

环境搭建

Node.js:

[https://nodejs.org/zh-cn]: https://nodejs.org/zh-cn “https://nodejs.org/zh-cn“

Git:

[https://git-scm.com/downloads]: https://git-scm.com/downloads “https://git-scm.com/downloads“

下载 Node.js 和 Git 程序并安装，一路点 “下一步” 按默认配置完成安装。

连接 Github

注册github账号（

[https://github.com/]: https://github.com/ “https://github.com/“

），选择免费账户，并验证邮件。

右键—Git Bash Here—设置用户名和邮箱：

创建 SSH 密匙

输入

然后一路回车

添加密匙

进入 C:\Users\用户名\.ssh目录(如果没有找到，就勾选隐藏的项目)，用记事本打开公钥 id_rsa.pub 文件并复制里面的内容。

![](C:\Users\90922\Pictures\Screenshots\屏幕截图 2023-07-04 115457.png)

登陆 GitHub ，点自己的头像，进入 Settings（设置） 页面，选择左边栏的 SSH and GPG keys（SSH 和 GPG 密钥），点击 New SSH key（新的SSH密钥）。

Title（标题） 随便填，粘贴复制的 id_rsa.pub 内容到 Key 中，点击 Add SSH key（添加SSH密钥） 完成添加。

![](C:\Users\90922\Pictures\Screenshots\屏幕截图 2023-07-04 121204.png)

验证连接

打开 Git Bash Here，输入 ssh -T git@github.com 出现 “Are you sure……”，输入 yes 回车确认。

显示 “Hi xxx! You’ve successfully authenticated, but GitHub does not provide shell access.” 即连接成功。

创建 Github Pages 仓库

GitHub 主页右上角加号 -> New repository（新存储库）：

  • Repository name（存储库名称） 中输入 用户名.github.io
  • 勾选 Initialize this repository with a README（添加自述文件）
  • Description 选填

填好后点击 Create repository（创建存储库） 创建。

![img](C:\Users\90922\Downloads\image (1).png)

Hexo 博客文件夹目录结构如下：

|-_config.yml # 网站的配置信息

|-package.json # 应用程序的信息

|-scaffolds # 模板文件夹

|-source # 存放用户资源，Markdown文档

| |-_drafts

| |-_posts

|-themes # 主题文件夹

|-public # 网站文件
